# 🎓 Calphark Cognify - Premium EdTech Platform

A high-end, gamified educational technology platform built with React, featuring AI-powered learning experiences, interactive lessons, and comprehensive progress tracking.

## ✨ Features

### 🎨 Premium UI/UX Design
- **Soft Pastel Gradients** - Beautiful color schemes throughout
- **Glassmorphism Cards** - Modern frosted glass effects
- **Large Rounded Components** - 20-28px border radius for contemporary look
- **3D Illustrations & Mascots** - Engaging visual elements
- **Smooth Micro-interactions** - Polished hover animations and transitions
- **Mobile-First Responsive** - Perfectly optimized for all devices

### 📱 Three-Layer Layout System

#### 1. Top Navigation Bar
- App logo with animated interactions

- Notification center with badges

  - 🏆 Achievements

- Animated hover highlights
- Daily challenge widget

#### 3. Main Dashboard

**Welcome Section**

**Gamified Progress System**
- Circular XP progress ring

## 🛠️ Tech Stack

- **React** - UI framework
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Advanced animations
- **Lucide React** - Beautiful icons
- **Vite** - Build tool and dev server

## 🚀 Getting Started

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Access the App

Open your browser and navigate to:
```
http://localhost:5173
```

## 🎯 Design Philosophy

This platform follows modern SaaS design principles with:

1. **Professional Polish** - No AI-generated or template feel
2. **Consistent Design System** - Unified spacing, typography, and colors
3. **Performance First** - Optimized animations and interactions
4. **Accessibility** - Semantic HTML and ARIA labels
5. **Responsive Design** - Mobile-first approach
